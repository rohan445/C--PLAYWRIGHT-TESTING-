# 🧪 SauceDemo Automation Testing (C# Playwright)

## 📌 Project Overview

This project contains an automated end-to-end test cycle for the SauceDemo application using **C#** and **Playwright**.

The test simulates a real user journey:

* Login
* Select product
* Add to cart
* Checkout
* Complete order

---

## 🌐 Application Under Test

* Website: https://www.saucedemo.com/
* Type: E-commerce demo application

---

## 🛠️ Tech Stack

* Language: C#
* Framework: Playwright (.NET)
* IDE: VS Code / Visual Studio
* Shell: PowerShell

## 🧪 Test Scenario (End-to-End Flow)

### ✅ Steps Covered

1. Navigate to SauceDemo website
2. Login using valid credentials

   * Username: `standard_user`
   * Password: `secret_sauce`
3. Add product to cart
4. Navigate to cart
5. Proceed to checkout
6. Enter user details:

   * First Name
   * Last Name
   * Postal Code
7. Complete the order

### ✅ Expected Result

* User successfully completes purchase
* Order confirmation message is displayed
